Place your datasets in this folder before running the app.

1. LinkedIn Job Postings Dataset
   - Source: https://www.kaggle.com/datasets/arshkon/linkedin-job-postings
   - After downloading, preprocess using the notebooks provided
   - Save final file as: company_cluster.csv

2. Udemy Courses Dataset
   - Source: https://www.kaggle.com/datasets/andrewmvd/udemy-courses
   - After downloading, preprocess using the notebooks provided
   - Save final file as: ufinal.csv

Note: These files are not included in the repo due to large file sizes
(company_cluster.csv = ~32MB, ufinal.csv = ~199MB)
